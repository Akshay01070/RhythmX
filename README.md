# RhythmX

RhythmX is a static landing page for a smartwatch product. This repository contains the front-end source for the RhythmX marketing site — a responsive static site that showcases product features, specifications, pricing, testimonials, and an email signup form.

Contents:
- `index.html` — main markup and structure
- `styles.css` — styling and layout
- `script.js` — interactive behaviors (theme toggle, mobile nav, sliders, forms)

Quick start:
- Open [index.html](index.html) in your browser.
- Or run a local server:

```bash
python -m http.server 8000
# or
npx http-server .
```

Notes:
- This is a static landing page intended for demos and marketing. Replace assets and copy as needed for production deployment.
